Yoga Pose Detector

About This Project

Welcome to the Yoga Pose Detector! This project uses computer vision and machine learning to identify various yoga poses in real-time. The system employs OpenCV, MediaPipe, and Flask to provide an interactive experience.

Features:

  • Real-time yoga pose detection
  • Support for multiple yoga poses
  • Timer functionality to track the duration of poses
  • User-friendly web interface

Explore and improve your yoga practice with our intelligent Yoga Pose Detector!
